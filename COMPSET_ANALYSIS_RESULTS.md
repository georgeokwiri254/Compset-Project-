## Appendix C: Glossary of Terms

| Term | Definition |
|------|------------|
| **ADR** | Average Daily Rate - average revenue per occupied room |
| **BAR** | Best Available Rate - publicly offered rate with no restrictions |
| **Compset** | Competitive Set - group of hotels used for performance comparison |
| **RevPAR** | Revenue Per Available Room - ADR Ã— Occupancy % |
| **RPM** | Revenue Per Mille (thousand) - revenue per 1,000 available rooms |
| **Index** | Relative performance vs. compset (1.00 = fair share, >1.00 = beating market) |
| **RGI** | Revenue Generation Index - RevPAR index vs. compset |
| **MICE** | Meetings, Incentives, Conferences, Events - group business segment |
| **FIT** | Free Independent Traveler - individual bookings (not groups) |
| **OTA** | Online Travel Agency - Booking.com, Expedia, etc. |
| **STR** | Smith Travel Research - industry benchmark data provider |
| **Ceiling Rate** | Maximum rate charged (typically for peak demand periods) |
| **Floor Rate** | Minimum rate charged (typically for distressed inventory) |
| **Compression** | Difference between ceiling and floor rates (higher = more variability) |
| **Overlap Score** | Percentage of business mix similarity between properties |
| **Business Mix** | Distribution of revenue across customer segments |
